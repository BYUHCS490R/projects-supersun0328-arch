Internet

The Internet is a global network that connects computers and devices so they can communicate and share information.

Web

The Web is a system of websites and web pages that people access through the Internet using a browser.

How are the Internet and Web different?

The Internet is the network itself, while the Web is a service that runs on the Internet and allows users to view websites.

ASCII and Unicode

ASCII is a basic character encoding system that represents English letters and symbols. Unicode is a larger system that supports characters from many languages around the world.

What is UTF-8

UTF-8 is a type of Unicode encoding that is commonly used on the web and supports many characters while saving space

URL

A URL is the address used to locate a resource on the Internet, such as a webpage.

Domain Name

A domain name is the human-readable name of a website that represents an IP address.

HTTP

HTTP is a protocol that allows browsers and servers to communicate and transfer data on the web.

HTTPS

HTTPS is a secure version of HTTP that encrypts data to protect user information.

DNS

DNS is a system that translates domain names into IP addresses so computers can find websites.

W3C Recommendations

W3C Recommendations are standards created to ensure the web works consistently across different browsers and devices.

XML, XHTML, HTML, HTML5

XML is used to store and transport data.

HTML is used to create web pages.

XHTML is a stricter version of HTML.

HTML5 is the latest version of HTML with more features for modern web applications.
