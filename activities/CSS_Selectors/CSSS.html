What is the difference between an ID and a Class?

Use ID for one specific thing (like a main header) abd use Class for styles you repeat.

What is the difference between a SPAN tag and a DIV tag?

DIV = big container & SPAN = small, inline styling

What is the difference between a DIV tag and a HEADER, MAIN, or FOOTER?

  1. div = no specific meaning
  2. header - top of a page of section
  3. main - main body of the page
  4. footer - bottom of the page like copyright ©
